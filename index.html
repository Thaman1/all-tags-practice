Introduction to Git and Github

Git is a software that is used to track different changes made codebase overtime.Changes can be tracked offline and update are shared by pushing or pulling from a remote repositorysuch as Github

Github is a cloud base platform designed to help developers manage, host and collaborate on projects using the Git version control system.

Basic Git commands

  1. Git init
  2. Git add .
  3. Git status
  4. Git commit -m
  5. Git push
